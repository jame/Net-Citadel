Net-Citadel

Debian packaging for Net::Citadel

This project is primarily about Debian packaging for Net::Citadel, which is a Perl module that is a "first cut on supporting the Citadel application protocol". This repository was initially forked from gitpan/Net-Citadel.git and the debian branch was then added to be used for the development of the debian packaging directory for the Perl module.

The name of the Debian package is libnet-citadel-perl and it is being done under the auspices of the Debian Perl Team. The status of the package there can be seen by going to their package tracking page and then searching for the package name.
