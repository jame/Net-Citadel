Net::Citadel is a Perl module which can be used to access the Citadel.org application protocol using Perl. Besides its development and availibility here, distribution of the module can also be found at CPAN and in Debian.

This project was initially forked from the Net::Citadel CPAN mirror gitpan/Net-Citadel.git (represented now by the gitpan branch here) and the debian branch was then added to the repository where it and the master branch (and along with the pristine-tar branch) are used for the development and maintenance of the Debian packaging. Development of the module itself is done on the develop branch and stable releases are kept on the stable branch.

The module is packaged for inclusion in Debian as libnet-citadel-perl, where it is being done under the auspices of the Debian Perl Team. The status of the package itself can be seen at its Package page. Other information for it can be seen by going to the pkg-perl package tracking page and then searching for the package name.
